# User Management REST API with Flask

## Description
A simple REST API using Flask that allows you to add, view, update, and delete user data stored in an in-memory Python dictionary.

## Requirements
- Python 3.x
- Flask (`pip install flask`)
- Requests (`pip install requests`; for client testing)

## Files
- `app.py`: Flask server code containing all CRUD endpoints
- `client.py`: Script using requests to test the API

## How to Run
1. Start the Flask server:
2. In a new terminal, test using the client script:

## API Endpoints
- `GET /users` - View all users
- `POST /users` - Add a new user
- `PUT /users/<user_id>` - Update a user
- `DELETE /users/<user_id>` - Delete a user
